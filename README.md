# 🐾 WildLens — Animal Encyclopedia (Free Gemini Version)

AI-powered animal encyclopedia using Google Gemini Flash — completely FREE!

## Quick Setup

### 1. Install dependencies
```
pip install -r requirements.txt
```

### 2. Get your FREE Gemini API key
- Go to: https://aistudio.google.com
- Sign in with Google
- Click "Get API Key" → "Create API key"
- Copy it

### 3. Set your API key (PowerShell in VS Code)
```
$env:GEMINI_API_KEY="your-key-here"
```

### 4. Run the app
```
python app.py
```

### 5. Open browser
```
http://localhost:5000
```

## That's it! Upload any animal photo and click Get Info 🎉
